# Ricoh Recipes

This tool supports to generate the readable files from a JSON file.

## Usage

- Put the JSON input file under `assets`.
- Input the environment variables
- run `yarn install`
- run `yarn build`

## Structure

```
|_ assets
| |_ input-recipes.json
|_ dist
| |_ db/output-recipes.json
|_ utils
|_ index.js
```
